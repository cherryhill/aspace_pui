For WG: Mark suggested the effect that is demonstrated if you hover over pager 4. Note that this would only be in place for Title, Date Added and Date Modified sorts, not for the Relevance sort. For Brian: While this seems like a good suggestion, we are unsure of what would be required to program it, and so if the WG likes this suggestion, we look to you to let us know if it can be implemented in Phase 1, or if this is a later project.

For developer: On displays that lists numerous records, the contents of the teaser text would be the abstract note field. In the absence of content in this field, it would be the initial 200-300 characters from the scope & contents note field. If both of these are empty, then it would inherit from the record above it in the hierarchy. If there is no data to display, then that field simply doesn't display.
