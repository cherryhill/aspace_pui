ArchivesSpace PUI Wireframes

These are functional wireframes containing the suggested PUI enhancements for ArchivesSpace. For more details on this project, and the process we used to get here, please see the Public Interface Enhancement Project - ArchivesSpace documentation.

Keep in mind that none of this is real ArchivesSpace functionality. Clicking on links will only take you to examples of the resulting display.
